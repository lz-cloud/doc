# Ollama

## Install

### download

- https://ollama.com/download
- ollama 支持的所有模型: https://ollama.com/library


### ollama 命令
| 命令          | 用途      |
|-------------|---------|
| ollama      | 使用帮助    |
| ollama list | 显示模型列表  |
| ollama show | 显示模型的信息 |
| ollama pull | 拉取模型    |
| ollama push | 推送模型    |
| ollama cp   | 拷贝一个模型  |
| ollama rm   | 删除一个模型  |
| ollama run  | 运行一个模型  |

### Install on Mac

- install:
  - unzip and move to Application
  - run:  `ollama run qwen2.5:7b` # 无则下载，有则安装
  - stp: `/bye`
