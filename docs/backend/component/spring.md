# Redis

> 此模块聚合了 spring 各种组件，和spring 的小工具，方便快速集成 spring

## 依赖
```xml
<dependency>
    <groupId>com.wkclz.spring</groupId>
    <artifactId>shrimp-cloud-spring</artifactId>
</dependency>
```

## 配置
```
spring: 此空间下的所有
```

## 依赖信息

- web
- aop
- freemarker
- mail
- actuator
- prometheus
- jasypt
- 其他未拆解的依赖
