# 选型

## 选型前提
1. vue框架【国产】
2. 新技术栈，让自己不断学习，努力活过35岁
3. 组件，demo齐全。容易找到解决方案

## 分析
- Element 最多，但基本停留在 VUE2
- RuoYi-Vue3 没有使用 TS
- Vben Admin 整体技术栈比较新，适合试点平台，增强生存能力

故选型为：***Vben Admin***
