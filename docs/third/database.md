# 数据库

## 用途
> 数据存储

## 对接
> 阿里云 RDS 服务(可以不购买，直接在服务器上安装，需要更强大的服务器，数据可靠性较弱)

## 配置
1. 可以选择不购买，在服务器上安装，如果数据不算重要
2. 独立购买数据库，可以有很高的性能，数据可靠性也大大增强(若服务体量很大，请不要心疼钱)
3. 购买时长越长，单价越便宜
