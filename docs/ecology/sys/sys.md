# SYS 系统管理模块


## 模块定位
 系统配置，系统级数据 

## 数据库表

- sys_access URI白名单
- sys_access_log 接口调用日志
- sys_area 地区表【已废弃，前端独立实现即可】
- sys_category 分类信息
- sys_config 系统配置
- sys_dict_type 字典类型
- sys_dict 字典
- sys_email_template 邮件模板
- sys_error 系统异常日志
- sys_file 系统文件上传记录
- sys_rdm  研发任务管理
- sys_tenant 租户
- sys_tenant_seo 租户SEO
